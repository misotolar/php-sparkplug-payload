# PHP Sparkplug Payload

A library that provides tools for encoding and decoding payload objects using
the Sparkplug Google Protocol Buffer Schema described in the Sparkplug
Specification from Sparkplug Specification Project Team.

https://sparkplug.eclipse.org/specification/version/3.0/documents/sparkplug-specification-3.0.0.pdf

## Installation

```console
composer require misotolar/php-sparkplug-payload
```
